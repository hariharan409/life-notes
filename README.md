## ⚡ Quick Start: Adding a New Tool Note

1. Go to `tools/` folder
2. Duplicate `_templates/tool-note.md`
3. Rename it to the tool name (e.g. `git.md`, `ffmpeg.md`)
4. Fill in the sections
5. Add it to `tools/_index.md`
